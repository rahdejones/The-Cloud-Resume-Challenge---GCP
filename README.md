# The-Cloud-Resume-Challenge - GCP
A full-stack DevOps project featuring a serverless resume website hosted on Google Cloud Platform. Demonstrates proficiency in GCP infrastructure, CI/CD pipelines, and infrastructure-as-code (Terraform)

☁️ Cloud Resume Challenge — Google Cloud Edition
This repository contains the source code and infrastructure configuration for my personal cloud resume. This project was built to demonstrate end-to-end cloud engineering skills, moving from a static frontend to a fully automated, serverless backend on Google Cloud Platform.

🛠️ The Tech Stack
Frontend: HTML5, CSS3, JavaScript (Hosted on Cloud Storage)

Global Delivery: Cloud CDN & Load Balancing (HTTPS/SSL)

Backend API: Cloud Functions (Python/Go/Node.js)

Database: Firestore (NoSQL) for visitor counter tracking

Infrastructure as Code: Terraform

CI/CD: GitHub Actions (Automated testing and deployment)
